# Oriflame

A Flutter prototype for a TikTok/Reels-style vertical scrolling feed of shoppable posts with AI-generated captions and quick-share functionalities.

## How to Run the Project

1. **Ensure Flutter is installed:** Make sure you have the Flutter SDK installed (version `>=3.3.0`). You can verify your environment by running `flutter doctor`.
2. **Install Dependencies:** Navigate to the project directory in your terminal and fetch the required packages:
   ```bash
   flutter pub get
   ```
3. **Run the App:** Launch the application on a connected physical device or emulator by running:
   ```bash
   flutter run
   ```
